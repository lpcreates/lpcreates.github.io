PacMan Factory

This application uses a factory function to create multiple PacMen on screen via the "Add PacMan" button. The PacMen start idle on screen until the "Start Game" button is activated. The PacMen will travel the screen randomly, bouncing off the edges. If the "Start Game" button is activated again, the PacMen speed up.
